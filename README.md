# Password Strength Checker

A simple web application that checks the strength of a password in real time.
It helps users create secure passwords by verifying common security requirements.

## Features

* Password strength indicator
* Show / hide password option
* Strong password generator
* Requirement checklist for secure passwords
* Responsive and simple user interface

## Password Requirements

The password must contain:

* At least **8 characters**
* **One uppercase letter**
* **One number**
* **One special character**

The strength bar updates automatically when the user types a password.

## Technologies Used

* **HTML5** – Page structure
* **CSS3** – Styling and layout
* **JavaScript** – Password validation and strength calculation

## How to Use

1. Open the project in a browser.
2. Enter a password in the input field.
3. The system checks password strength instantly.
4. Use **Generate Strong Password** to create a secure password.
